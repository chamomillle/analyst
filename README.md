# analyst
# Анализ процесса реструктуризации кредитов

**Автор:** Марина Сергеева

**Цель:** провести анализ процесса реструктуризации, рассчитать ключевые показатели, построить прогноз и визуализировать результаты.

---

## 📊 Результаты

| Показатель | Значение |
|------------|----------|
| Всего заявок | 11 031 |
| Уровень одобрения | 21.0% |
| Среднее время на решение | 100.6 часов |
| Конверсия одобренных в сделки | 79.2% |
| Прогноз на март-май 2021 | ~3 200 заявок в месяц |

**Ключевые выводы:**
1. **Просрочка — главный фактор отказа:**  
   заявки с бакетом 181+ одобряются в 4 раза реже (5.6%), чем без просрочки (24.5%).

2. **Кредитные карты — слабое звено:**  
   конверсия одобренных в сделку — всего 41.6% (против 80%+ у других продуктов).

3. **Январь 2021 — аномальный пик:**  
   на этот месяц пришлось 75% всех заявок (8 312 из 11 031).  
   **Рекомендация:** усилить ресурсы в пиковые месяцы.
