##  О проекте

Этот проект — комплексный анализ процесса реструктуризации кредитов. Основная цель — выявить узкие места в процессе, оценить эффективность текущей модели принятия решений и предложить конкретные шаги для улучшения.

##  Главные выводы и рекомендации для бизнеса

### 1. Просрочка — главный фактор отказа  
Заявки с просрочкой (пакет 181+) одобряются в **4 раза реже** (5.6% против 24.5%).  
**Рекомендация:** пересмотреть скоринг для клиентов с небольшими просрочками. Это может увеличить уровень одобрения на **15-20%** без роста рисков.

### 2. Кредитные карты — слабое звено  
Конверсия одобренных в сделку — всего **41.6%** (против 80%+ у других продуктов).  
**Рекомендация:** провести аудит процессов коммуникации с клиентами на этапе одобрения. Возможно, клиенты не понимают условия или теряются в процессе. Улучшение может дать **+30% сделок** в этом сегменте.

### 3. Январь 2021 — аномальный пик  
В январе поступило **75% всех заявок** (8 312 из 11 031). Это создало большую нагрузку на команду.  
**Рекомендация:** усилить ресурсы в пиковые месяцы, автоматизировать часть ручных проверок. Это сократит время на решение и повысит качество обслуживания.

## 📈 Прогноз на март-май 2021

На основе анализа трендов ожидается **~3 200 заявок в месяц**. Это позволяет заранее спланировать ресурсы и избежать перегрузки команды.

##  Как запустить проект

1. Склонируйте репозиторий:  
   `git clone https://github.com/chamomillle/analyst.git`

2. Установите зависимости:  
   `pip install pandas openpyxl sqlite3`

3. Откройте `analit.ipynb` и выполните ячейки по порядку.

4. Для дашборда откройте `analitik.pbix` в Power BI Desktop.
